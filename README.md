# 🚀 Clozzy Support Center

![Clozzy Support](https://img.shields.io/badge/Clozzy-Support%20Center-ff3860)
![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

A modern, responsive, and secure support center for the **Clozzy** private couple space app. Built with vanilla JavaScript, Firebase Firestore, and a stunning dark theme with pink and blue accents.

![Preview](assets/preview.png)

## ✨ Features

- 🔍 **Smart Search** - Search across all help articles with debounced input
- 📂 **Category Filters** - Filter by app features (Daily Selfie, Chat, Gallery, etc.)
- 📝 **Ticket System** - Submit support requests with email for account verification
- 📱 **Fully Responsive** - Optimized for mobile, tablet, and desktop
- 🎨 **Modern UI** - Black, white, pink & blue theme with glassmorphism effects
- ⚡ **Fast Performance** - Debounced search, cached Firebase queries
- 🔒 **Secure** - Environment variables for Firebase config, XSS protection
- ♿ **Accessible** - ARIA labels, keyboard navigation, screen reader support
- 📊 **Analytics** - Optional event tracking for user behavior
- 🌐 **Offline Support** - Graceful fallback when Firebase is unavailable

## 🎯 Supported Features

Help articles and support for all Clozzy features:

- 📸 **Daily Selfie** - Selfie sending, streaks, notifications
- 💬 **Chat** - Message delivery, reactions, media sharing
- 🖼️ **Gallery** - Photo loading, sharing, permissions
- 📅 **Timeline** - Entry ordering, date filters
- 📊 **Stats** - Dashboard updates, data sync
- 🎨 **Theme** - Custom themes, premium features
- 📝 **Note** - Cloud sync, sharing
- 😊 **Mood Status** - Updates, freezing issues
- ⚡ **Quick Actions** - Custom shortcuts
- 🔐 **Sign Up/Login** - Authentication, 2FA, password reset
